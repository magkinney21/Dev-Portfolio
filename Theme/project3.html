PROJECT DETAILS



About This Project

DevHops is an app that allows users to sign up, create, and delete an account in order to add reviews of their favorite beers. Once posted on the app, the user reviews are then stored on a database, allowing other users to see all reviews. When reviewing a beer, each user has the ability to rate it based upon the beers name, type, brewery, alcohol percent,pairing by language, as well as upload their favorite image.

THE GOAL

For this project, we collaborated to make an application using the MEAN stack. We worked in a team of four. Using pair-programming techniques. So we would swap pairs at least once a day so that everyone had a chance to work on each part of the application. Team : Magdalena Kinney, Dan Milo, Liam Robinson, Michael Torres ​



THE RESULT

A working full-stack application, built by our team. This app has 3 models – one representing someone using your application (a user), and two that represent the main functional ideas of our app. This app uses Angular as a front-end framework with RESTful routes. Along with, mongo queries using the Mongoose module to interact with our database. We used promises to make our lifes easier. Has semantically clean HTML and CSS

THE IDEA

The idea for this app was to have a place where developers could come and review beers. It's known that developers love beer and tend to have pretty good taste. So why not make a beer review catered to developers! Theres even an option to sort by pairings, incase you want to see what beer goes best with javascript or etc.

UPCOMING FEATURES

For our upcoming features for this app, we will be implementing a third party api to pull in beer so you are able to search. We plan on cleaning up the design some to be more uniform and make it sufficient for all devices. Stay tuned!
